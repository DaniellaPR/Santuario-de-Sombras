# El Libro de las Reliquias Perdidas
Experiencia web interactiva de estilo gótico-fantástico donde el usuario explora un grimorio mágico, libera entidades atrapadas y resuelve desafíos simbólicos para revelar un mensaje oculto.

## Descripción:

El Libro de las Reliquias Perdidas es una experiencia narrativa interactiva desarrollada con HTML, CSS y JavaScript puro.

El usuario es guiado a través de un libro encantado donde debe activar reliquias, interactuar con entidades mitológicas y completar una serie de minijuegos para recolectar fragmentos de una palabra final.

Cada reliquia representa una “bestia” con identidad visual, sonora y mecánica propia, transformando la navegación en una experiencia progresiva, inmersiva y dinámica.

## Características:

- Experiencia interactiva basada en narrativa
- Minijuegos únicos por cada reliquia:
  - Ritual de velas (tiempo y precisión)
  - Liberación de alas (interacción progresiva + fases)
  - Ruta de navegación (decisiones encadenadas)
  - Secuencia de memoria direccional (tipo patrón)
- Sistema de ambientes dinámicos:
  - cambio de fondo por reliquia
  - ambientación visual y sonora independiente
- Sistema de audio contextual:
  - música de fondo por escena
  - efectos de interacción (hover, error, recompensa)
- Elementos visuales dinámicos:
  - niebla animada
  - partículas brillantes
  objetos voladores interactivos
- Sistema de progreso:
  - recolección de letras
  - desbloqueo de cofre final
- Final interactivo:
  - ingreso de palabra clave
  - revelación de mensaje personalizado


## Tecnología:

- HTML5
- CSS3 (animaciones, layout, efectos visuales)
- JavaScript (DOM, eventos, lógica de juego)
- Web Audio API básica (Audio)


## Lógica principal:

1. Portada → activación de experiencia
2. Selección de reliquia → cambio de ambiente
3. Interacción con entidad → narrativa + minijuego
4. Resolución → obtención de letra
5. Progreso completo → desbloqueo de cofre
6. Validación de palabra → revelación final

Cada reliquia tiene:
- ambientación visual
- pista de audio
- mecánica interactiva


## Arte visual:

-Estética gótica elegante

-Paleta: negro, morado, rojo profundo, detalles suaves luminosos

-Inspirado en catedrales, bestiarios mitológicos, grimorios
    
-Uso de contraste entre oscuridad y elementos brillantes
